# Software Studio Story

A mobile game where you play as a Scrum Master leading a software development team. Manage sprints, balance priorities, and keep your team happy while delivering software on time.

## Features

- Sprint simulation with story points and velocity tracking
- Drag-and-drop Kanban board for managing work items
- Team morale system affecting productivity
- Event-driven gameplay with stakeholder demands and technical challenges

## Tech Stack

- React Native with Expo
- React Native Reanimated for animations
- React Native Gesture Handler for drag-and-drop
- Zustand for state management

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Run on Android
npm run android

# Run on iOS
npm run ios

# Run on web
npm run web
```

## How to Play

1. **Start a Sprint** - Each sprint runs for a set number of days
2. **Manage the Board** - Drag story tickets from Todo to Doing to Done
3. **Balance Work** - Complete stories to earn points, but don't let the team burn out
4. **Handle Events** - Respond to bugs, scope creep, and stakeholder requests
5. **Win the Sprint** - Deliver maximum value without crashing morale
